# Compiling Instructions

## Cloning from GitHub

In order to clone from GitHub, it is advisable to clone using the
`--recurse-submodules` flag to automatically initialise submodules.

```
git clone --recurse-submodules git@github.com:pocket-boy/pocket-editor.git
```

## Compilation

All dependencies are included in with the project under the `lib/` directory.

When any updates are made to the `CMakeLists.txt` or during initial compilation,
set up the build environment using the following command:

```
cmake -B build
```

For all modifications to source code or during initial compilation after the build
environment has been set up, run the following:

```
cmake --build build
```

The resulting executable will be built into the `build/` directory generated by
CMake.
